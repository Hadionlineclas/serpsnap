== CLOUDFLARE PE WEBSITE UPLOAD KARNE KA TARIKA ==
(Step by Step — Super Easy)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 1: ACCOUNT BANAO
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. cloudflare.com pe jao
2. "Sign Up" karo — free hai, koi card nahi chahiye
3. Email verify karo

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 2: PAGES PE JAO
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. Login ke baad left sidebar mein "Workers & Pages" dikhega
2. Click karo
3. "Pages" tab select karo
4. "Create a project" button click karo
5. "Direct Upload" option select karo (GitHub ki zaroorat nahi)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 3: FILES UPLOAD KARO
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. Project ka naam rakho: serpsnap (ya jo pasand ho)
2. "Upload assets" area mein SAARI files drag & drop karo:
   - index.html      ← Main tool page
   - blog.html       ← Blog page
   - about.html      ← About page
   - contact.html    ← Contact page
   - privacy-policy.html ← Privacy page (AdSense ke liye ZARURI)
3. "Deploy site" button click karo
4. 30 seconds mein site live ho jayegi!

Teri URL hogi: serpsnap.pages.dev

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 4: UPDATES KAISE KARO (Future mein)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. Pages > serpsnap > Deployments
2. "Create new deployment" click karo
3. Updated files upload karo
4. Deploy — bas itna!

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 5: ADSENSE APPLY KARO (1-2 mahine baad)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. adsense.google.com pe jao
2. "Get started" click karo
3. Apni website URL dalo: https://serpsnap.pages.dev
4. Apna Gmail (hadionlineclas@gmail.com) se apply karo
5. Diya hua verification code index.html ke <head> mein paste karo
6. Re-upload karo Cloudflare pe
7. "Verify" click karo — Google 2-4 haftay mein review karega

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

2. "Add property" → URL: https://serpsnap.pages.dev

4. Tag ko index.html ke <head> mein paste karo, re-upload karo
5. Verify karo
6. "URL Inspection" → apna URL dalo → "Request Indexing"

Yeh karne se Google 2-3 din mein index kar leta hai!
